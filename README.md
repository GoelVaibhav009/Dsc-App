# DSC, Chitkara University 
Official App for Developer Student Clubs, Chitkara University

#### Start Learning 
* [Documenatation](https://flutter.dev/docs)

## To get Started

  ##### Clone the repository
```bash
git clone https://github.com/Yashank18/Dsc-App.git
```
  ##### Move to the folder
```bash
cd \dscchitkara
```

  ##### To run the app,
```bash
flutter run
```

## To Contribute
